# Music_Thoughts ♫

Площадка для оценки и обсуждения музыки на русском языке. Стилизация — ранний интернет (1996-2001).

## Что внутри

- **Сайт** на Flask + SQLite (всё локально, без внешних зависимостей кроме pip-пакетов).
- **Десктопная версия** (`.exe` под Windows / бинарник под macOS/Linux) через pywebview + PyInstaller — открывает тот же сайт в нативном окне.
- **Три роли**:
  - `viewer` (зритель) — читает рецензии, ставит +/-, комментирует, отправляет жалобы.
  - `reviewer` (рецензент) — всё, что viewer, + создаёт/редактирует свои рецензии.
  - `moderator` (модератор) — полный контроль над сайтом (см. ниже).
- **Гостевая книга** (классика жанра), бегущая строка, счётчик посещений.

## Установка

```bash
cd Music_Thoughts
python3 -m venv .venv
source .venv/bin/activate         # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск веб-сайта

```bash
python app.py
```

Открыть в браузере: <http://127.0.0.1:5000>.

База данных создаётся автоматически (файл `music_thoughts.db`) при первом запуске. Туда же seed'ятся:

- 3 модератора, 1 рецензент, 1 зритель;
- 4 демонстрационные рецензии;
- одна запись в гостевой книге;
- стандартные настройки и список запрещённых слов.

## Запуск настольного приложения (dev-режим)

```bash
python desktop_app.py
```

Откроется отдельное окно `Music_Thoughts.exe` с встроенным сайтом.

## Сборка `.exe`

На Windows (даёт `Music_Thoughts.exe`):
```bash
pip install pyinstaller pywebview
python build_exe.py
```

Готовый файл — в `dist/Music_Thoughts/Music_Thoughts.exe`. Папка `dist/Music_Thoughts/` целиком переносимая.

На macOS/Linux PyInstaller соберёт нативный бинарник в той же папке. Чтобы получить именно `.exe`, сборку нужно делать на Windows-машине (или через Wine / GitHub Actions с windows-runner).

## Возможности модератора

Кнопка `[ КОНСОЛЬ МОДА ]` в верхнем меню (видна только модераторам). Внутри:

- **Сводка** — счётчики жалоб, пользователей, рецензий, комментариев, забаненных.
- **Пользователи** — поиск, смена роли (viewer/reviewer/moderator), бан/разбан с причиной, удаление.
- **Рецензии** — скрытие, добавление в «Рекомендуем», правка, удаление.
- **Комментарии** — последние 200, удаление.
- **Жалобы** — обработка с указанием результата.
- **Журнал действий** — все модераторские операции с автором и временем.
- **Статистика** — роли, жанры, топ авторов, посещения.
- **Настройки сайта** — бегущая строка, MOTD, открытость регистрации/гостевой, фильтр запрещённых слов (звёздочки в новых текстах).
- **Гостевая книга** — модератор может скрывать записи прямо со страницы.

## Заметки по безопасности

- Пароли хранятся как `werkzeug.security.generate_password_hash`.
- Модераторы могут менять роли и удалять пользователей, но не могут забанить/удалить самих себя.
- Все действия модерации логируются в `mod_log` с автором, временем и деталями.
- Для production переменную `MT_SECRET` в окружении надо задать заранее (используется как `app.secret_key`).
